# Cliftonville Website

A modern, professional multi-page website for **Cliftonville** - A premier supported living community for adults (18+) in Ogun State, Nigeria.

## Features

- **Multi-page Architecture** - Separate pages for Home, About, Services, Facilities, and Contact
- **Responsive Design** - Mobile-first approach, looks great on all devices
- **Smooth Animations** - Powered by Framer Motion for a world-class feel
- **Brand-Consistent Design** - Uses Cliftonville's brand colors (Royal Blue & Orange)
- **Professional UI** - Clean, modern design with excellent UX

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool for fast development
- **Tailwind CSS v4** - Utility-first CSS framework
- **React Router v6** - Client-side routing
- **Framer Motion** - Animation library
- **Lucide React** - Modern icon set

## Pages

1. **Home** - Hero section, core values, feature highlights
2. **About** - Mission, vision, values, team section
3. **Services** - Care plans, 24/7 support, medication management, etc.
4. **Facilities** - Recreation, wellness, dining amenities with stats
5. **Contact** - Referral form, contact information, map placeholder

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Chidi09/Clifton-ville-website.git

# Navigate to the project
cd Clifton-ville-website

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Building for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
